Sources column links only to primary legal/regulatory text (statute pages, regulator websites, parliamentary bill records). Media section gathers third-party analysis.

Status: Implemented = in force and being enforced · Passed = enacted but not yet in force · Pending = in legislative process · Stalled = died, blocked, or stuck. Type: Full ban = age-based access prohibition · Parental consent = requires guardian approval · Design rules = duty of care / addictive features rules · Age verification = age assurance / age verification only · Mixed = combination of two or more regulatory approaches. Litigation: Active = open challenge · Enjoined = currently blocked by court · Struck down = permanently invalidated · None = no pending challenge / not applicable.

Methodology. This tracker was built using primary legislative and regulatory sources. Each jurisdiction entry links directly to the official statute, bill, or regulator page. Jurisdiction coverage on social media age verification issues was cross-referenced against the Tech Policy Press Global Social Media Age Restriction Tracker. Litigation data draws on the NetChoice Litigation Center. Age verification legislation coverage draws on the AVPA State Laws Tracker. The tracker is updated weekly via an automated process that monitors legislative and regulatory developments, court decisions, and commentary.
